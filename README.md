# Banking App (Java CLI)

This is a simple Java command-line application that accepts customer details as arguments and prints an account summary.

## Requirements
- JDK 17 installed
- Terminal / Command Prompt

## Compile the Program
Open terminal in the project folder and run:

javac BankingApp.java

## Run the Program
Provide the following arguments:
customerName accountNumber accountType balance

Example:

    Input : java BankingApp Mithul 12345 Savings 5000
    
    Output : Account Summary
            Customer Name : 12345
            Account Number : Savings
            Current Balance : 5000.00
